# Python Playground

This repository serves as a playground for exploring various Python topics, patterns, and libraries. Each subdirectory contains a self-contained project or example set focusing on a specific concept.

## Projects

### Dependency Injection (`di`)

A comprehensive guide to dependency injection in Python. It demonstrates the evolution from hardcoded dependencies to manual injection and finally using a DI container. It also includes a FastAPI integration example.
